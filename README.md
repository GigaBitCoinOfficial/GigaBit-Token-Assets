  <h2>1. Introduction</h2>
  <p>
    GigaBit Coin (GBC) is a BEP-20 token built on Binance Smart Chain (BSC) that powers a community-first decentralized ecosystem. GBC is focused on providing a reward structure tied to real user actions such as adding liquidity, staking, and long-term holding. Our mission is to grow the ecosystem responsibly while enabling DeFi use cases with verifiable participation.
  </p>
